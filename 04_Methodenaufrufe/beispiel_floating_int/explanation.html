Erklärung

  • Aufruf von FloatingWorld.show():

    Diese Methode wird aufgerufen, die FloatingWorld anzuzeigen. Sie ist statisch und gehört zur Klasse FloatingWorld.

  • Aufruf von FloatingWorld.addPlayer():

    Fügt einen neuen Spieler zur FloatingWorld hinzu. Die Methode ist statisch und wird direkt über den Klassennamen aufgerufen.

  • Aufruf von FloatingWorld.movePlayerLeft(0):

    Diese Methode bewegt einen bestimmten Spieler nach links. Auch diese Methode ist statisch und kann direkt über den Klassennamen aufgerufen werden. In diesem Beispiel entspricht 0 dem Spieler, der am längsten in der Welt existiert.
