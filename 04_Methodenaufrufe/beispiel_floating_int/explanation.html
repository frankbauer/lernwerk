Erklärung

  • Aufruf von FloatingWorld.show():

    Diese Methode wird aufgerufen, um die FloatingWorld anzuzeigen. Sie ist statisch und gehört zur Klasse FloatingWorld und wird somit über den Klassennamen aufgerufen.

  • Aufruf von FloatingWorld.addPlayer():

    Diese Methode fügt einen neuen Spieler zur FloatingWorld hinzu. Sie ist statisch und wird ebenfalls direkt über den Klassennamen aufgerufen.

  • Aufruf von FloatingWorld.movePlayerLeft(0):

    Diese Methode bewegt einen, per Pramater ausgewählten, Spieler nach links. Auch diese Methode ist statisch und kann direkt über den Klassennamen aufgerufen werden. In diesem Beispiel entspricht 0 dem Spieler, der am längsten in der Welt existiert.
