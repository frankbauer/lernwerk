Erklärung

Die Klasse Kreis implementiert zwei Schnittstellen: Form2D und Verortbar.

  • Interface Form2D:

    public double berechneFlaeche();
    Dieses Interface deklariert eine Methode zur Berechnung der Fläche einer zweidimensionalen Form. Jede Klasse, die Form2D implementiert, muss daher ebenso eine öffentliche Methode berechneFlaeche definieren.

  • Interface Verortbar:

    public void setLocation(double x, double y);
    Dieses Interface deklariert eine Methode zur Festlegung der Position eines Objekts. Jede Klasse, die Verortbar implementiert, muss daher ebenso eine öffentliche Methode setLocation definieren.

  • Klasse Kreis:

    public class Kreis implements Form2D, Verortbar
    Die Klasse Kreis implementiert sowohl die Form2D als auch die Verortbar Schnittstelle. Sie muss daher beide Methoden berechneFlaeche und setLocation definieren.

    Jedes Kreis Objekt hat die drei Attribute radius, x und y, die den Radius des Kreises und seine Position speichern.

    • Methode berechneFlaeche:

      public double berechneFlaeche()
      Diese Methode berechnet die Fläche des Kreises anhand seines Radius. Die Formel lautet: this.radius * this.radius * Math.PI.

    • Methode setLocation:

      public void setLocation(double xIn, double yIn)
      Diese Methode setzt die Position des Kreises, indem sie die Parameter xIn und yIn den entsprechenden Attributen der Klasse zuweist.

    • Konstruktor Kreis:

      public Kreis(double radius)
      Dieser Konstruktor initialisiert den Kreis mit einem angegebenen Radius.

    • Methode toString:

      public String toString()
      Diese Methode überschreibt die Standard-toString-Methode und gibt eine Beschreibung des Kreises als String aus, die den Radius und die Position des Kreises enthält..

  • Hauptmethode main:

    Die Hauptmethode dient als Einstiegspunkt für die Ausführung des Programms. Hier werden drei Objekte der Klasse Kreis erstellt und in Variablen unterschiedlicher Typen gespeichert. Das Beispiel soll zeigen, welche Datentypen für die Referenzvariablen verwendet werden können, wenn ein Objekt einer Klasse erstellt wird, die mehrere Schnittstellen implementiert.

    • Kreis k = new Kreis(2);
      Ein Kreisobjekt mit einem Radius von 2 wird erstellt und in einer Variablen vom Typ Kreis gespeichert.
    • Form2D f = new Kreis(4);
      Ein Kreisobjekt mit einem Radius von 4 wird erstellt und einer Form2D Variablen zugewiesen. Da die Klasse Kreis das Form2D Interface implementiert, kann ein Kreisobjekt als Form2D behandelt werden. Über f können nur Methoden aufgerufen werden, die in Form2D deklariert sind. Der Aufruf f.setLocation(0, 0) ist also zum Beispiel ungültig.
    • Verortbar v = new Kreis(6);
      Ein Kreisobjekt mit einem Radius von 6 wird erstellt und einer Verortbar Variablen zugewiesen. Da die Klasse Kreis das Verortbar Interface implementiert, kann ein Kreisobjekt als Verortbar behandelt werden. Über v können nur Methoden aufgerufen werden, die in Verortbar deklariert sind. Der Aufruf v.berechneFlaeche() ist also zum Beispiel ungültig.
