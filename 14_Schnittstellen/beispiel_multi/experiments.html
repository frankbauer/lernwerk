Schreiben Sie eine Klasse Ellipse mit Flächeninhalt A = π r a r b , die ebenfalls die beiden Schnittstellen Form2D und Verortbar implementiert.
Erweitern Sie die Schnittstelle Verortbar um eine Methode, mit der die y-Position des Objektes ausgelesen werden kan. Wenn Sie die Schnittstelle verändern, dann müssen auch alle Klassen die diese implementieren angepasst werden. 
interface Verortbar {
    public void setLocation(double x, double y);
    public double getX();
    }
Schreiben Sie in der Klasse App eine Methode, die die x-Position eines verortbaren Objektes auf den Wert des Flächeninhalts eines zweiten Objektes setzt und die y-Position beibehält. Ihre Methode sollte einen Parameter mit dem Typ Verortbar und einen für den Typ Form2D haben. 
static void move(Verortbar v, Form2D form){
    v.setLocation(form.berechneFlaeche(), v.getY());
