Aufgabe

Implementieren Sie die untenstehende Methode has so, dass geprüft wird, ob ein Knoten mit dem Wert needle vom Knoten cur aus erreichbar ist. Das Ergebnis soll zurückgegeben werden. Sie können dafür die Instanzmethoden int getPayload() und List<Node<Integer>> childNodes() der Klasse Node<Integer> verwenden. Es kann von einem zyklenfreien Graphen ausgegangen werden.
