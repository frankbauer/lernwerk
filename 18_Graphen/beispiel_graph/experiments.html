Versuchen Sie, den Graphen zu erweitern. Es können wie zuvor Knoten mit g.add() und Kanten mit g.addEdge() hinzugefügt werden.
Schreiben Sie eine Methode, mit der Sie eine ungerichtete Kante zwischen zwei Städten hinzufügen können. Eine ungerichtete Kante zwischen den Knoten A und B entspricht zwei Kanten, eine von A nach B und eine von B nach A. Die Methodensignatur könnte so aussehen: 
public static void addEdge(Graph<String> g, String a, String b) {}
Da Sie die neue Methode nicht in der Klasse Graph sondern in der Klasse SimpleGraph schreiben, schreiben Sie die Methode am besten als Klassenmethode (static). Die Methode könnte so aussehen: 
public static void addEdge(Graph<String> g, String a, String b) {
    g.addEdge(a, b);
    g.addEdge(b, a);
}
Passen Sie Ihren Code so an, dass Sie Kantengewichte angeben, die der Entfernung in km entsprechen. Die Signatur Ihrer addEdge Methode muss angepasst werden, um das Gewicht zu übergeben. Ihre neue Methodensignatur könnte wie folgt aussehen: 
public static void addEdge(Graph<String> g, String a, String b, double weight)
 Die angepasste Methode könnte dann so aussehen:
public static void addEdge(Graph<String> g, String a, String b, double weight){
    g.addEdge(a, b, 660);
    g.addEdge(b,a,660);
}
Ein Methodenaufruf könnte dann so aussehen:
SimpleGraph.addEdge(g, "Berlin", "Amsterdam", 660);
