Geben Sie für jeden Knoten im Graphen den Ausgangsgrad (= Anzahl der ausgehenden Kanten) aus. Mit .childNodes() bekommen Sie eine Liste mit allen Kindknoten. Die Größe dieser Liste entspricht dem Ausgangsgrad. 
for (Node<String> n : g) {
    System.out.println(n + ": " + n.childNodes().size());
}
Geben Sie für jeden Knoten außerdem noch den Eingangsgrad (= Anzahl der eingehenden Kanten) aus. Der Eingangsgrad lässt sich nicht direkt ablesen. Sie müssen dazu für jeden Knoten alle Knoten besuchen, und zählen wie häufig der gesuchte Knoten als Kindknoten vorkommt. 
for (Node n : g) { 
    int count = 0;
    for (Node k : g) {
        for (Node child : k.childNodes()){
            if (child == n) count++;
        }
    }
    //Ausgabe im Format: Knoten: Ausgangsgrad, Eingangsgrad
    System.out.println(n + ": " + n.childNodes().size() + ", " + count);
}
