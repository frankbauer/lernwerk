Geben Sie für jeden Knoten im Graphen die Ausgangsgrad (= Anzahl der ausgehenden Kanten) an. Mit .childNodes() bekommen Sie eine Liste mit allen Kindknoten. Die Größe dieser Liste entspricht dem Ausgangsgrad. 
System.out.println(n + ": " + n.childNodes().size());
Geben Sie für jeden Knoten außerdem noch den Eingangsgrad an. Der Eingangsgrad lässt sich nicht direkt ablesen. Sie müssen dazu für jeden Knoten alle Knoten besuchen, und zählen wie häufig der gesuchte Knoten als Kindknoten vorkommt. 
int count = 0;
for (Node other : g) {
    for (Node child : other.childNodes()){
        if (child == n) count++;
    }
}
System.out.println(n + ": " + n.childNodes().size() + ", " + count);
