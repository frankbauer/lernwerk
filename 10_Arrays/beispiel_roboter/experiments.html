Ändern Sie das Programm so, dass mit einer Schleife 10 Roboter an zufälligen Positionen angelegt werden. Legen Sie ein Feld vom Typ Robot in der entsprechenden Größe an und füllen Sie es in einer Schleife. Um eine zufällige Position im Raum zu wählen können Sie vom Room die verfügbare Breite w und Höhe h abfragen und so zum Beispiel die Position für jeden Roboter bestimmen:
x = ⌊ random() ⋅ ( w - 200 ) + 100 ⌋
y = ⌊ random() ⋅ 40 + ( h - 50 ) ⌋ 
Robot[] robots = new Robot[10];
final int wd = Room.getRoomWidth();
final int hg = Room.getRoomHeight();

for (int i = 0; i < robots.length; i++) {
    final int x = (int)(Math.random() * (wd - 2 * 100) + 100);
    final int y = (int)(Math.random() * 40 + (hg - 50));
    
    robots[i] = new Robot(x, y, livingRoom);
}
Ergänzen Sie das Programm so, dass die Positionen der 10 Roboter nicht zufällig, sondern durch zwei weitere int-Felder vordefiniert werden. Eines der Felder speichert pro Roboter die zugehörige x-Position, das andere die y-Position. Zugehörig heißt in diesem Fall, dass die x-Position an der gleichen Stelle im Feld steht wie die y-Position im anderen un der Roboter im robots-Feld. 
Robot[] robots = new Robot[10];
int[] xPos = {103, 169, 235, 301, 367, 433, 499, 565, 631, 697};
int[] yPos = {439, 427, 413, 429, 408, 434, 416, 419, 437, 408};

for (int i = 0; i < robots.length; i++) {
    robots[i] = new Robot(xPos[i], yPos[i], livingRoom);
