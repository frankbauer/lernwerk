Erklärung

Dieses Beispiel demonstriert die Verwendung eines Arrays zum Speichern und Verwalten mehrerer Roboterobjekte.

  • Erstellung und Initialisierung des Robot-Arrays

    Ein Array namens robots wird verwendet, um eine feste Anzahl von referenzen auf Roboter-Instanzen zu speichern. In diesem Fall hat das Array eine Länge von zwei, was bedeutet, dass es zwei Roboterreferenzen aufnehmen kann:

    Robot[] robots = new Robot[2];

    Der Zugriff auf einen Eintrag aus dem Array ist wie der Zugriff auf eine Variable mit dem Datentyp Robot. Speziell ist robots[0] aus Ausdruck dessen Ergebnistyp Robot sein wird. Der Wert ist die am Index 0 gespeicherte Referenz.

  • Platzierung der Roboter im Raum

    Jeder Roboter wird individuell mit spezifischen Koordinaten und einem Verweis auf das gleiche Room-Objekt (Wohnzimmer) initialisiert. Dies zeigt, wie Objektreferenzen in einem Array gespeichert werden können:

    • Der erste Roboter wird bei den Koordinaten (690, 410) platziert. 
      robots[0] = new Robot(690, 410, livingRoom);
    • Der zweite Roboter wird bei den Koordinaten (260, 430) platziert. 
      robots[1] = new Robot(260, 430, livingRoom);

    Steht ein Ausdruck wie robots[0] links von einer Zuweisung, so wird der Wert der zugewiesene Wert an der Stelle 0 im Array gespeichert. In diesem Beispiel ist der Wert eine Referenz auf ein neues Robot-Objekt, das bei den Koordinaten (690, 410) im Wohnzimmer platziert wird.

  • Aktivierung der Bewegung der Roboter

    Eine einfache For-Schleife durchläuft das Array und aktiviert die Bewegung jedes Roboters. Dies verdeutlicht, wie man über die Elemente eines Arrays iterieren und Aktionen auf jedes Element anwenden kann:

    for (int i = 0; i < robots.length; i++) {
    robots[i].setMoving(true);
}
