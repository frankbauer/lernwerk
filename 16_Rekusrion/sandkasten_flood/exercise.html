Aufgabe

In dieser Aufgabe sollen Sie eine Flood-Fill-Funktion implementieren. Flood Fill ist ein Algorithmus, der häufig in Grafik- und Zeichenanwendungen verwendet wird, um eine zusammenhängende Fläche zu füllen, die von einem Startpunkt (im Beispiel die Spalte 10 in Zeile 3) aus erreichbar ist.

Der Algorithmus überprüft eine Zelle an der Position (Zeile, Spalte). Wenn diese noch nicht gefüllt ist, soll sie gefüllt werden und die Suche in allen Nachbarzellen rekursiv wiederholt werden. Implementieren Sie dazu die Methode farbEimer.

Zum Testen können Sie die Instanzmethode public boolean istGefuellt(int zeile, int spalte) der Klasse Leinwand verwenden. Um eine Zelle zu füllen, kann public void fuellen(int zeile, int spalte) verwendet werden. Mit den Instanzattributen int zeilen und int spalten können Sie die Größe der Leinwand abrufen.
