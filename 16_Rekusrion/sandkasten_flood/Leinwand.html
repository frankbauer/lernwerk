Beschreibung

Die Klasse Leinwand repräsentiert eine Leinwand, die aus einer bestimmten Anzahl von Zeilen und Spalten besteht. Sie ermöglicht das Überprüfen und Füllen von Zellen sowie das Festlegen einer zeitlichen Verzögerung für Befehle.

Konstruktoren

  • public Leinwand(String[] args)
    Erstellt eine neue Leinwand mit den angegebenen Dimensionen und initialisiert die Zellen basierend auf den übergebenen Argumenten.

Methoden

  • public void setDelay(int delayInMs)
    Setzt die Verzögerung für Befehle in Millisekunden. Der Wert wird auf mindestens 1 Millisekunde gesetzt.
  • public boolean istGefuellt(int r, int c)
    Überprüft, ob die Zelle an der angegebenen Position gefüllt ist. Gibt true zurück, wenn die Zelle gefüllt ist, und false, wenn sie nicht gefüllt ist. Bei ungültigen Zellenkoordinaten wird eine Fehlermeldung im Textfeld ausgegeben.
  • public void fuellen(int r, int c)
    Füllt die Zelle an der angegebenen Position. Bei ungültigen Zellenkoordinaten wird eine Fehlermeldung im Textfeld ausgegeben.

Attribute

  • public final int zeilen
    Die Anzahl der Zeilen auf der Leinwand.
  • public final int spalten
    Die Anzahl der Spalten auf der Leinwand.
