Beschreibung

Die Klasse Node repräsentiert einen Knoten in einem Graphen. Jeder Knoten speichert einen generischen Wert von Type T als Nutzlast (payload) und eine Liste von Kanten (children), die die Verbindungen zu anderen Knoten darstellen.

Konstruktoren

  • public Node(T payload)
    Erstellt einen neuen Knoten mit der angegebenen Nutzlast.

Methoden

  • public T getPayload()
    Gibt die Nutzlast dieses Knotens zurück.
  • public List<Node<T>> childNodes()
    Gibt eine Liste der Kindknoten zurück.
  • public Edge<T> addEdge(Node<T> target)
    Fügt eine neue Kante zum aktuellen Knoten hinzu, die auf einen Zielknoten zeigt und das Gewicht auf 1.0. setzt
  • public Edge<T> addEdge(Node<T> target, double weight)
    Fügt eine neue Kante zum aktuellen Knoten hinzu, die auf einen Zielknoten zeigt und ein Gewicht hat.
