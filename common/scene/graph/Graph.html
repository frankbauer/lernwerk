Beschreibung

Die Klasse Graph repräsentiert einen generischen Graphen, der Knoten und Kanten enthält, wobei eine Kante immer zwischen zwei Knoten liegt. Diese Klasse implementiert das Iterable-Interface, wodurch es möglich ist, über alle Knoten im Graphen mit einer ForEach-Schleife zu iterieren.

Konstruktoren

  • public Graph()
    Initialisiert einen neuen leeren Graphen.

Methoden

  • public Node<T> add(T payload)
    Fügt einen neuen Knoten mit der gegebenen Nutzlast zum Graphen hinzu und gibt diesen Knoten zurück.
  • public void add(T... payload)
    Fügt mehrere Knoten mit den gegebenen Nutzlasten zum Graphen hinzu.
  • public Edge<T> addEdge(T n1, T n2)
    Fügt eine Kante mit einem Standardgewicht von 1.0 zwischen zwei Knoten mit den Nutzlasten n1 und n2 hinzu, wenn diese existieren.
  • public Edge<T> addEdge(T n1, T n2, double weight)
    Fügt eine Kante mit einem spezifischen Gewicht zwischen zwei Knoten mit den Nutzlasten n1 und n2 hinzu.
  • public Edge<T> addEdge(Node<T> source, Node<T> target)
    Fügt eine Kante mit einem Standardgewicht von 1.0 zwischen die spezifizierten Knoten source und target hinzu.
  • public Edge<T> addEdge(Node<T> source, Node<T> target, double weight)
    Fügt eine Kante mit einem spezifischen Gewicht zwischen die spezifizierten Knoten hinzu.
  • public Node<T> find(T payload)
    Sucht und gibt den Knoten zurück, der die gegebene Nutzlast trägt, oder null falls kein Knoten gefunden wurde.
  • public void visualize()
    Visualisiert den Graphen.

Überschriebene Methoden

  • public java.util.Iterator<Node<T>> iterator()
    Ermöglicht die Iteration über alle Knoten im Graphen, indem ein Iterator zurückgegeben wird.
