Beschreibung

Die Klasse TreeLibrary stellt eine Sammlung von vordefinierten Bildobjekten dar, die verschiedene Darstellungen von Bäumen in unterschiedlichen Jahreszeiten enthalten. Diese Bilder können für das Rendern von Baumobjekten verwendet werden.

Felder

  • public static final Image SPRING
    Ein Bild, das einen Baum im Frühling darstellt.
  • public static final Image SUMMER
    Ein Bild, das einen Baum im Sommer darstellt.
  • public static final Image AUTUMN
    Ein Bild, das einen Baum im Herbst darstellt.
  • public static final Image WINTER
    Ein Bild, das einen Baum im Winter darstellt.
  • public static final Image LATE_AUTUMN
    Ein Bild, das einen Baum im Spätherbst darstellt.
  • public static final Image POW
    Ein Bild, das eine besondere Grafik darstellt.
  • public static final Image STAGE
    Ein Bild für den Hintegrund (die Bühne).
