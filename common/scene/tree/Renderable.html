Beschreibung

Das Interface Renderable beschreibt eine Vereinbarung für Objekte, die gezeichnet werden können. Jedes Alle Klassen, die dieses Interface implementieren, besitzen die nachfolgenden Methoden.

Methoden

  • void render(Graphics2D g)
    Wird aufgerufen, wenn der Bildschirminhalt neu gezeichnet werden soll. g ist der dafür verfügbare Zeichenkontext.
