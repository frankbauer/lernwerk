Beschreibung

Die Klasse Graphics2D repräsentiert eine 2D-Grafik-Rendering-Engine. Sie verwaltet eine Liste von Render-Objekten und ermöglicht das Zeichnen von Bildern auf einer grafischen Oberfläche. Diese Klasse verwendet einen CommandBuffer zur Verwaltung der Zeichenbefehle.

Konstruktoren

  • Kein öffentlicher Konstruktor verfügbar. Verwenden Sie die Methode instance(), um die Singleton-Instanz (also einzige verfügbare Instanz) zu erhalten.

Methoden

  • public static Graphics2D instance()
    Gibt die Singleton-Instanz von Graphics2D zurück. Erstellt die Instanz, falls sie noch nicht existiert.
  • public void registerForRender(Renderable r)
    Registriert ein neues Render-Objekt zur Rendering-Liste.
  • public void render()
    Rendert alle registrierten Objekte, indem die render()-Methode jedes Objekts aufgerufen wird.
  • public void drawImage(Image img, int x, int y)
    Zeichnet ein Bild an die angegebene Position (x, y) in der Originalgröße.
  • public void drawImage(Image img, int x, int y, double scale)
    Zeichnet ein Bild an die angegebene Position (x, y) mit der angegebenen Vergrößerung scale.
  • public void drawImage(Image img, int x, int y, double sx, double sy)
    Zeichnet ein Bild an die angegebene Position (x, y) mit den angegebenen Vergrößerungen sx und sy.
