Beschreibung

Diese Klasse verwaltet eine spielerbasierte Umgebung, in der Spieler hinzugefügt, entfernt und bewegt werden können.

Methoden

  • public static void show()
    Zeigt die Welt an.
  • public static Player addPlayer()
    Fügt einen neuen Spieler zur Welt hinzu und gibt das Spielerobjekt zurück.
  • public static void removePlayer(int idx)
    Entfernt den Spieler am spezifizierten Index aus der Welt, falls der Index gültig ist.
  • public static void moveLeft()
    Bewegt die gesamte Welt nach links.
  • public static void moveRight()
    Bewegt die gesamte Welt nach rechts.
  • public static void move(int x)
    Bewegt die gesamte Welt um den angegebenen Wert in horizontaler Richtung.
  • public static void movePlayerLeft(int idx)
    Bewegt einen spezifischen Spieler nach links, basierend auf dem angegebenen Index.
  • public static void movePlayerRight(int idx)
    Bewegt einen spezifischen Spieler nach rechts, basierend auf dem angegebenen Index.
  • public static void movePlayer(int idx, int x)
    Bewegt einen spezifischen Spieler nach um den angegebenen Wert in horizontaler Richtung.
