Beschreibung

Diese Klasse repräsentiert einen Spieler in einer virtuellen Welt. Jeder Spieler hat einen spezifischen Typ (ein Aussehen) und kann sich innerhalb der Welt bewegen.

Konstruktoren

  • public Player()
    Erstellt einen neuen Spieler des Typs 0 und fügt ihn zur Welt hinzu.
  • public Player(int type)
    Erstellt einen neuen Spieler mit einem spezifizierten Typ und fügt ihn zur Welt hinzu.

Öffentliche Methoden

  • public void moveLeft()
    Bewegt den Spieler um einen festen Wert nach links.
  • public void moveRight()
    Bewegt den Spieler um einen festen Wert nach rechts.
  • public void move(int value)
    Bewegt den Spieler um den angegebenen Wert.
