Beschreibung

Die Klasse Sonne repräsentiert eine Sonne in einem Universum und ist für die Verwaltung einer Sammlung von Planeten zuständig.

Konstruktoren

  • public Sonne()
    Erstellt eine neue Sonne ohne zugeordnetes Universum.
  • public Sonne(Universum u)
    Erstellt eine neue Sonne und ordnet sie einem spezifischen Universum zu. Wenn das Universum null ist, wird diese Sonne nicht zugeordnet.

Öffentliche Methoden

  • public void print()
    Druckt Informationen über die Sonne und ihre Planeten ohne Einrückung. Diese Methode ist eine Überladung von print(String indent), die mit einem Standardwert für eine leere Einrückung aufgerufen wird. Die Methode ist damit eine Kurzschreibweise für print("").
  • public void print(String indent)
    Druckt Informationen über die Sonne und alle zugehörigen Planeten, wobei für jeden Planeten eine zusätzliche Einrückung hinzugefügt wird, um die hierarchische Struktur zu verdeutlichen. Ist indent null, wird die Einrückung auf einen leeren String ("") gesetzt.
