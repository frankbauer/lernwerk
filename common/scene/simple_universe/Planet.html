Beschreibung

Die Klasse Planet repräsentiert einen Planeten, der einer Sonne zugeordnet ist und Eigenschaften wie Radius und Geschwindigkeit besitzt.

Konstruktoren

  • public Planet()
    Erstellt einen neuen Planeten ohne zugeordnete Sonne. Ruft den anderen Konstruktor mit null als Argument auf.
  • public Planet(Sonne s)
    Erstellt einen neuen Planeten und ordnet ihn einer spezifischen Sonne zu. Wenn die Sonne nicht null ist, wird dieser Planet automatisch bei der gegebenen Sonne registriert.

Öffentliche Methoden

  • public void setRadius(double d)
    Setzt den Radius des Planeten auf den angegebenen Wert.
  • public void setSpeed(double d)
    Setzt die Umlaufgeschwindigkeit des Planeten auf den angegebenen Wert.
  • public String toString()
    Gibt eine Zeichenkettendarstellung des Planeten zurück, die Geschwindigkeit und Radius beinhaltet. Dies Methpde überschreibt das Standardverhalten wenn eine Planeten-Instanz in eine Zeichenkette (zum Beispiel bei der Verwendung mit System.out.println) umgewandelt werden soll.
  • public void print()
    Druckt Informationen über den Planeten ohne Einrückung. Diese Methode ist eine Überladung von print(String indent), die mit einem Standardwert für eine leere Einrückung aufgerufen wird. Die Methode ist damit eine Kurzschreibweise für print("").
  • public void print(String indent)
    Druckt Informationen über den Planeten, wobei indent als Einrückung verwendet wird, um die Ausgabe zu strukturieren. Ist indent null, wird es auf einen leeren String ("") gesetzt.
