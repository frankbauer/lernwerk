Beschreibung

Die Klasse Room stellt einen Raum in einer simulierten Umgebung dar. Räume können sichtbar oder unsichtbar sein und verwalten eine Liste von Robotern, die in ihnen operieren.

Konstruktoren

  • public Room()
    Erstellt einen neuen Raum, der standardmäßig sichtbar ist.
  • Room(boolean visible)
    Erstellt einen neuen Raum. Der übergebene Parameter bestimmt ob der Raum nach der Erstellung sichtbar ist. Der Raum wird.

Methoden

  • public static void displayMainRoom(boolean display)
    Steuert die Anzeige des Hauptzimmers und setzt dessen Sichtbarkeit.
  • public static int getRoomHeight()
    Gibt die Höhe aller Räume zurück.
  • public static int getRoomWidth()
    Gibt die Breite aller Räume zurück.
  • public void show(boolean display)
    Legt fest an ob der Raum angezeigt werden soll oder nicht.
