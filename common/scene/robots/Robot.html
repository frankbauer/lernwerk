Beschreibung

Die Klasse Robot repräsentiert einen Roboter, der in verschiedenen Räumen innerhalb einer simulierten Umgebung platziert und bewegt werden kann. Jeder Roboter hat einen eindeutigen Typ.

Konstruktoren

  • public Robot()
    Erstellt einen neuen Roboter mit einem automatisch zugewiesenen Typ.
  • Robot(int type)
    Erstellt einen Roboter mit einem spezifischen Typ.
  • public Robot(int x, int y, Room room)
    Erstellt einen Roboter, setzt seine Anfangsposition auf die gegebenen Koordinaten und weist ihm einen Raum zu.

Methoden

  • public void setLocation(int x, int y)
    Setzt die Position des Roboters auf die angegebenen Koordinaten.
  • public void setRoom(Room room)
    Weist den Roboter einem spezifischen Raum zu und aktualisiert den Raumstatus entsprechend. Diese Methode wirft eine Ausnahme, wenn der Raum null ist.
  • public void setMoving(boolean moving)
    Aktualisiert den Bewegungsstatus des Roboters.
