Beschreibung

Die Klasse RobotFactory dient als zentrale Fabrik zur Erstellung und Steuerung von Robotern in einer simulierten Umgebung.

Konstruktoren

Die Klasse enthält keine öffentlichen Konstruktoren, da alle Methoden statisch sind und direkt über die Klassenreferenz aufgerufen werden.

Öffentliche Methoden

  • public static Robot addRobotToMainRoom(int x, int y)
    Erstellt einen neuen Roboter, setzt seine Position und weist ihn dem Hauptzimmer zu.
  • public static void startMovingRobots()
    Startet die Bewegung aller Roboter im Hauptzimmer.
  • public static void stopMovingRobots()
    Stoppt die Bewegung aller Roboter im Hauptzimmer.
