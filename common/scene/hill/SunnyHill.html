Beschreibung

Die Klasse SunnyHill repräsentiert einen Hügel in einer simulierten Umgebung.

Konstruktoren

  • public SunnyHill(String[] args)
    Erstellt eine neue Instanz eines SunnyHill. Dieser Konstruktor nimmt ein Array von Strings entgegen, für das die Variable args (Parameter der main-Methode) eingesetzt werden kann.
