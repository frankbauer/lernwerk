Beschreibung

Die Klasse Tree stellt einen Baum in einer simulierten Umgebung dar. Jedes Baumobjekt wird mit einem zufälligen Stil erstellt. Ein Baum kann auf einem Hügel gepflanzt werden.

Konstruktoren

  • public Tree()
    Erstellt einen neuen Baum mit einem zufällig gewählten Stil.

Methoden

  • public void plantAtHill(SunnyHill hill)
    Pflanzt den Baum auf einem spezifizierten Hügel, indem die Methode addTree des Hügels aufgerufen wird. Dies verknüpft den Baum effektiv mit dem Hügel innerhalb der simulierten Umgebung.
