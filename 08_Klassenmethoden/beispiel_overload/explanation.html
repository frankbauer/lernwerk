Erklärung

Die Klasse Overload demonstriert das Konzept der Methodenüberladung in Java. Methodenüberladung ermöglicht es, innerhalb einer Klasse mehrere Methoden mit demselben Namen, aber unterschiedlichen Parameterlisten zu definieren.

  • Definieren der Klasse:

    Die Klasse wird wie gewohnt mit public class Overload definiert.

  • Erste Methode square:

    public static int square(int x) ist eine Methode, die das Quadrat einer ganzen Zahl berechnet. Der einzige Parameter x ist vom Typ int. Mit der Zeile return x * x; gibt das Ergebnis von x multipliziert mit sich selbst zurück. Da der Parameter vom Typ int ist, ist auch das Ergebnis eine ganze Zahl.

  • Zweite Methode square:

    public static double square(double x) ist eine überladene Methode. Die Methode hat denselben Namen wie die erste Methode, aber einen Parameter vom Typ double. Mit der Zeile return x * x; wird auch hier das Quadrat der Zahl berechnet. Da der Parameter vom Typ double ist, ist aber in diesem Fall auch das Ergebnis eine Fließkommazahl.

  • Hauptmethode main:

    public static void main(String[] args) ist der Einstiegspunkt für die Anwendung.

    • System.out.println( Overload.square(3) ); ruft die erste Methode square auf, da ein int-Parameter übergeben wird. Das Ergebnis ist deshalb wieder eine ganze Zahl und die Ausgabe ist damit 9.
    • Da der Aufruf von System.out.println( Overload.square(3.0) ); einen double-Parameter verwendet, wird die zweite Methode square aufgerufen. Das Ergebnis ist eine Fließkommazahl und die Ausgabe ist deshalb 9.0.
