Erstellen Sie eine weitere Überladung für den Datentypen String. Wandeln Sie die übergebene Zeichenkette zunächst in eine Zahl um (siehe Java API). Rufen Sie diese Methode auf. Mit Double.parseDouble(String) können Sie eine Zeichenkette in eine Zahl umwandeln. 
public static double square(String str) {
    double x = Double.parseDouble(str);
    return x * x;
}
