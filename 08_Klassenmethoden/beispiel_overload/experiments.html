Erstellen Sie eine weitere Überladung welche einen Parameter des Datentypen String annimmt. Wandeln Sie die übergebene Zeichenkette zunächst in eine Zahl um (siehe Java API). Rufen Sie dannach die überladene Methode auf. Mit Double.parseDouble(String) können Sie eine Zeichenkette in eine Zahl umwandeln. 
public static double square(String str) {
    double x = Double.parseDouble(str);
    return x * x;
}
