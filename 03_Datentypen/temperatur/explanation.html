Erklärung

Dieses Java-Programm liest eine Temperatur in Grad Celsius von der Webseite ein und rechnet diese in Fahrenheit um. Das Ergebnis wird auf der Konsole ausgegeben.

  • public class Convert { - Deklariert die Klasse Convert, in der das Programm ausgeführt wird.
  • public static void main(String[] args) { - Definiert die main-Methode, die der Einstiegspunkt für das Programm ist. Ihr Programm führt nur das aus, was in der main-Methode steht oder von dort durch einen Methodenaufruf o.Ä. in Gang gesetzt wird. Sie bekommt beim Start (Klick auf "Ausführen") ein Array von Strings (args) als Parameter, welches die Programmargumente enthält. Im vorliegenden Fall enthält args[0] den Zahlenwert aus dem Eingabefeld oben als Zeichenkette (String). Der Aufruf der main-Methode mit den entsprechenden Parametern erfolgt automatisch im Hintergrund, sobald Sie auf "Ausführen" klicken.
  • final double celsius = Input.getTemperature(args); - Ruft die statische Methode getTemperature der Klasse Input auf, um die eingegebene Celsius-Temperatur zu lesen und weist sie einer konstanten (final) Variablen celsius zu. Konstant bedeutet, dass der Wert nach der Zuweisung nicht verändert werden kann. Im gegebenen Anwendungsfall ist es sinnvoll aber nicht verpflichtend, die Variable als konstant zu deklarieren.

    Die Methode getTemperature erwartet laut API (siehe Reiter "Input") ein String-Array als Parameter. Hierfür wird der Übergabeparameter der main-Methode args übergeben und damit die aus dem Textfeld eingelesene Zeichenkette mit der eingestellten Temperatur an die getTemperature-Methode weitergegeben. Die Methode liest nun aus dem übergebenen Argument die Temperatur aus und gibt diese als Zahlenwert (double) zurück.

  • double fahrenheit = celsius * 9 / 5 + 32; - Rechnet den Celsius-Wert celsius mittels arithmetischen Operationen entsprechend der gegebenen Formel in Fahrenheit um und speichert das Ergebnis in der neuen Variable fahrenheit. Die Variable celsius wird dadurch nicht verändert!
  • System.out.print(celsius + " Grad Celsius entsprechen "); - Gibt den Celsius-Wert gefolgt von einem Text (inklusive Leerzeichen) auf der Konsole aus, ohne einen Zeilenumbruch am Ende zu erzeugen (keine ln-Endung bei System.out.print).
  • System.out.println(fahrenheit + " Grad Fahrenheit."); - Gibt den umgerechneten Fahrenheit-Wert auf der Konsole aus und beendet die ausgegebene Zeile mit einem Zeilenumbruch (Endung ln).
