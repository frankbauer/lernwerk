Erklärung

Dieses Java-Programm liest eine Temperatur in Grad Celsius von der Webseite ein und rechnet diese in Fahrenheit um. Das Ergebnis wird auf der Konsole ausgegeben.

  • public class Convert { - Definiert die Klasse Convert, in der das Programm ausgeführt wird.
  • public static void main(String[] args) { Die main-Methode, die der Einstiegspunkt für das Programm ist. Sie Methode bekommt beim start ein Array von Strings (args) als Parameter, welcher die Programmargumente enthält. Im speziellen enthält args[0] die Zeichenkette aus dem Eingabefeld oben.
  • final double celsius = Input.getTemperature(args); Ruft die statische Methode getTemperature der Klasse Input auf, um die eingegebene Celsius-Temperatur zu lesen und weist sie einer konstanten Variablen celsius zu, was bedeutet, dass der Wert nach der Zuweisung nicht verändert werden kann.
  • double fahrenheit = celsius * 9 / 5 + 32; Rechnet den Celsius-Wert in Fahrenheit um.
  • System.out.print(celsius + " Grad Celsius entsprechen "); Gibt den Celsius-Wert gefolgt von einem Text auf der Konsole aus, ohne einen Zeilenumbruch am Ende zu erzeugen.
  • System.out.println(fahrenheit + " Grad Fahrenheit."); Gibt den umgerechneten Fahrenheit-Wert auf der Konsole aus und beendet die ausgegebene Zeile mit einem Zeilenumbruch.
