Erklärung

Dieses Programm demonstriert den Tausch von Werten zwischen zwei Variablen unter Verwendung einer temporären Hilfsvariablen.

  • Initialisierung der Variablen:

    Zwei double-Variablen, a und b, werden mit den Werten 42 bzw. 58 initialisiert.

    double a = 42;
double b = 58;
  • Verwendung einer Hilfsvariablen:

    Eine temporäre Variable temp wird verwendet, um den Wert von a zu speichern, bevor dieser überschrieben wird. Der Wert von temp ist nun 42.

    double temp = a;
  • Austausch der Werte:

    Der Wert von b wird a zugewiesen. Damit ist enthält a nun den Wert 58.

    Der ursprüngliche Wert von a, der im Moment in temp gespeichert ist, wird nun an b zugewiesen. Der neue Wert von b ist nun 42. Damit sind die Werte von a und b getauscht. 

    a = b;
b = temp;
  • Ausgabe der Ergebnisse:

    Die neuen Werte von a und b werden ausgegeben.

    System.out.println("a = " + a);
System.out.println("b = " + b);
