zusätzliches Experiment:

Können Sie den Wertetausch zweier Variablen auch ohne eine Hilfsvariable realisieren?

Der Wertetausch ohne eine Hilfsvariable kann durch die Verwendung von arithmetischen Operationen realisiert werden.

 
a = a + b;  //fuer Erklaerung im Kommentar sei initiales a = a' und initiales b = b'
b = a - b;  //b = (a' + b') - b' = a' (nach Aufloesen der Klammern bleibt initiales a uebrig)
a = a - b;  //a = (a' + b') - a' = b' (nach Aufloesen der Klammern bleibt initiales b uebrig)
