Erklärung

Diese Klasse implementiert eine Liste für zweidimensionale Vektoren mit einer festen Kapazität. Sie ermöglicht das Hinzufügen und Entfernen von Vektoren sowie die Berechnung der Summe aller gespeicherten Vektoren.

  • Array für Vektoren:

    Die Klasse verwendet ein Array vom Typ Vec2D[] zur Speicherung der Vektoren. Das Array wird als final deklariert, da die Referenz auf das Array sich nicht ändern soll. Die Länge des Arrays wird im Konstruktor festgelegt und kann danach nicht mehr verändert werden.

    public final Vec2D[] data;

    Das Array ist private (private), damit andere Klassen nicht direkt auf die gespeicherten Vektoren zugreifen können. Nicht belegte Positionen im Array enthalten den Wert null.

  • Konstruktor:

    Der Konstruktor erstellt ein neues Array mit der übergebenen Kapazität. Die Kapazität legt fest, wie viele Vektoren maximal gespeichert werden können. Alle Werte im Array werden automatisch mit null initialisiert. 

    public Vec2DList(int capacity){
    this.data = new Vec2D[capacity];
}

  • Methode sum():

    Diese Methode berechnet die Summe aller gespeicherten Vektoren:

    1. Initialisierung der Summenvariablen:

      Zwei separate Variablen werden für die Summen der x- und y-Koordinaten angelegt und mit 0 initialisiert.

      int xSum = 0;
int ySum = 0;
    2. Iteration und Addition:

      Eine Schleife durchläuft das komplette Array. Bei jedem nicht-null Eintrag werden die Koordinaten zu den jeweiligen Summen addiert.

      for (int i=0; i<this.data.length; i++){
    if (this.data[i] != null){
        xSum = xSum + this.data[i].x;
        ySum = ySum + this.data[i].y;
    }
}
    3. Rückgabe des Ergebnisvektors:

      Die berechneten Summen werden in einem neuen Vektor zusammengefasst und zurückgegeben.

      return new Vec2D(xSum, ySum);
  • Methode add():

    Die add-Methode fügt einen neuen Vektor in die Liste ein:

    1. Suche nach freiem Platz:

      Eine Schleife durchsucht das Array nach der ersten Position mit dem Wert null.

      for (int i=0; i<this.data.length; i++){
    if (this.data[i] == null){
    2. Einfügen des Vektors:

      An der gefundenen Position wird die Referenz auf den neuen Vektor gespeichert.

              this.data[i] = p;
    3. Beenden der Methode:

      Nach erfolgreichem Einfügen wird die Methode sofort beendet. Wurde kein freier Platz gefunden, wird der Vektor ignoriert.

              return;
    }
}
  • Methode delete():

    Die delete-Methode entfernt einen spezifischen Vektor aus der Liste:

    1. Suche nach dem Vektor:

      Eine Schleife durchsucht das Array nach einer Referenz auf den zu löschenden Vektor.

    if (this.data[i] == p){
    2. Referenzvergleich:

      Der Operator == vergleicht die Referenzen der Vektoren, nicht deren Inhalt. Es wird also nach genau dem übergebenen Vektorobjekt gesucht und nicht nach eine anderen Vektor, der dieselben Koordinaten spichert.

    3. Löschen des Vektors:

      Gefundene Positionen werden mit null überschrieben, wodurch die Referenz auf den Vektor entfernt wird.

              this.data[i] = null;
    }
