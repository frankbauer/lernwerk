Erklärung - Wald

Dieses Programm simuliert die Erstellung und Darstellung von Bäumen auf einer Bühne. Es demonstriert die grundlegende Nutzung von Objektorientierung, indem es Objekte der Klasse Baum verwendet und das Aussehen einzelner Instanzen über das Instanzattribut jahreszeit ändert.

  • Erstellen eines Baumes in der Mitte:

    Ein neues Baum-Objekt wird mittels dem Standardkonstruktor (keine Übergabeparameter) erstellt. Deshalb wird als Jahreszeit der Standardwert "herbst" gesetzt.

    Über die Variable mitte kann anschließend auf diesen Baum zugegriffen werden.

    Baum mitte = new Baum();
  • Erstellung und Festlegung der Jahreszeit des rechten Baumes:

    Ein weiteres Baum-Objekt wird mittels dem Parameterkonstruktor erstellt. Die Variable rechts referenziert die neue Instanz, das heißt darin befindet sich die Speicheradresse, über die auf die neue Instanz zugegriffen werden kann. Die Jahreszeit des Baumes wird bei der Erstellung auf den übergebenen Wert "winter" festgelegt.

    Baum links = new Baum("winter");
  • Erstellung und Festlegung der Jahreszeit des linken Baumes:

    Über den Parameterkonstruktor wird ein weiteres Baum-Objekt erstellt und eine Referenz darauf in der Variable links gespeichert. Die Jahreszeit des Baumes wird bei der Erstellung auf "spaetherbst" festgelegt. 

    Baum rechts = new Baum("spaetherbst");
  • Änderung der Jahreszeit des mittleren Baumes:

    Die Jahreszeit des Baumes, der durch die Variable mitte referenziert wird, wird zweimal geändert: zuerst auf "fruehling" und dann auf "fasching". Dies zeigt, dass Attribute (und damit der Zustand) von Objekten nach ihrer Erstellung geändert werden können. Der Zugriff auf das Attribut erfolgt stets mit [Name der Instanz].[Name des Attributs]. So kann mit de Attribut wie mit einer lokalen Variable gearbeitet werden.

    Achtung: Da der Zugriff auf das Attribut unkontrolliert erfolgt, kann auch eine ungültige Jahreszeit (im Beispiel "fasching") zugewiesen werden. Deshalb werden Attribute beim Schreiben einer neuen Klasse in der Regel als private deklariert und ein Zugriff aus anderen Klassen nur mittels Methoden (Getter/Setter) erlaubt.

    mitte.jahreszeit = "fruehling";
mitte.jahreszeit = "fasching";
