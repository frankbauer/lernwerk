Beschreibung

Die Klasse Baum repräsentiert einen Baum, der in verschiedenen Jahreszeiten dargestellt werden kann. Diese Klasse implementiert das Renderable-Interface, sodass Bäume gezeichnet werden können. Jeder Baum hat eine eindeutige ID und wird automatisch bei der Erstellung registriert, um gerendert (gezeichnet) zu werden.

Attribute

  • public String jahreszeit
    Die aktuelle Jahreszeit des Baumes. Gültige Jahreszeiten sind "winter", "fruehling", "sommer", "herbst" und "spaetherbst".

Konstruktoren

  • public Baum()
    Erstellt einen neuen Baum im Herbst. Der Baum wird automatisch zum Rendern registriert.
  • public Baum(String jahreszeit)
    Erstellt einen neuen Baum in der angegebenen Jahreszeit. Der Baum wird automatisch zum Rendern registriert.

Methoden

  • public void render(Graphics2D g)
    Zeichnet den Baum auf die Graphics2D-Oberfläche. Die Position und der Maßstab des Baums hängen von seiner ID ab. Der Baum wird entsprechend der angegebenen Jahreszeit gezeichnet.
