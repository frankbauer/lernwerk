Erweitere das Programm so, das gezählt wird wieviele Bäume gepflanzt wurden. Gebe die Anzahl der gepflanzten Bäume nach der Schleife aus. Erstelle eine Zählvariable count vor der Schleife und inkrementiere diese in jedem Schleifendurchlauf. Nach der Schleife kann der Wert der Zählvariable ausgeben werden. 
int count = 0;
while (gardener.hasSapling()) {
    gardener.plantTree();     
    count++;       
}
System.out.println("Gepflanzte Baeume: " + count);
Ändere das Programm so ab, dass der Gärtner maximal 5 der Setzlinge pflanzt. Das Programm muss zählen wieviele Bäume bereits gepflanzt wurden. Die Bedingung der while-Schleife muss diese Anzahl berücksichtigen. 
int count = 0;
while (gardener.hasSapling() && count<5) {
    gardener.plantTree();     
    count++;       
}
