Erklärung

Dieses Programm illustriert die Verwendung einer for-Schleife in Java, um mehrere Bäume auf einem Hügel zu pflanzen. Es zeigt wie eine Schleife verwendet werden kan um widerholt die selbe Sequenz von Anweisungen auszuführen.

  • Erstellen des Umgebungsobjekts:

    Ein Objekt der Klasse SunnyHill wird erstellt.

    SunnyHill hill = new SunnyHill(args);
  • Initialisierung und Ausführung der for-Schleife:

    Die for-Schleife wird eingerichtet, um den Code im Schleifenrumpf vier Mal zu durchlaufen. In jeder Iteration der Schleife wird ein Baum gepflanzt. Der Index i (wird in diesem Beispiel nicht verwendet) ändert sich in jedem Schleifendruchlauf (von 0 bis 3). 

    for (int i = 0; i < 4; i++) {
    • Erstellung und Pflanzung eines Baumes (Schleifenrumpf):

      In jedem Durchlauf der Schleife wird ein neues Baumobjekt (Tree) erstellt. Anschließend wird der Baum durch Aufrufen der Methode plantAtHill auf dem Hügel gepflanzt.

      Tree t = new Tree();
t.plantAtHill(hill);
    • Iteration und Schleifenende:

      Nachdem ein Baum gepflanzt wurde, beendet die Schleife den aktuellen Durchlauf und inkrementiert den Index i um eins (i++). Wenn i den Wert 4 erreicht, wird die Schleife beendet, da die Bedingung i < 4 nicht mehr erfüllt ist.
