Beschreibung

Die Klasse Recorder dient zur Aufzeichnung und Verwaltung von Messungen. Sie verwendet eine statische Liste von Measurement-Objekten, um Daten zu speichern und darauf zuzugreifen. Die Klasse bietet Methoden, um auf die gespeicherten Messwerte sequentiell zuzugreifen.

Methoden

  • public static boolean hasMore()
    Prüft, ob noch weitere Messungen in der Sammlung verfügbar sind. Gibt true zurück, wenn weitere Messungen vorhanden sind, andernfalls false.
  • public static Measurement getNext()
    Gibt die nächste Messung in der Sammlung zurück. Falls keine weiteren Messungen vorhanden sind, wird null zurückgegeben.
  • public static void reset()
    Setzt den Recorder zurück, sodass die Messungen von Anfang an erneut abgerufen werden können. Diese Methode sollte aufgerufen werden, wenn eine neue Durchsicht der Messwerte gestartet werden soll.
