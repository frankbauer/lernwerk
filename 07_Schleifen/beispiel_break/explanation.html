Erklärung

Dieses Programm demonstriert die Verwendung einer Endlosschleife mit bedingten Anweisungen für continue und break. Es verdoppelt fortlaufend einen Startwert, überspringt bestimmte Ausgaben und bricht die Schleife unter einer vorgegebenen Bedingung ab.

  • Initialisierung:

    Eine Variable i wird mit dem Wert 1 initialisiert.

    int i = 1;
  • Endlosschleife:

    Die Schleife startet ohne spezifische Abbruchbedingung (while (true)).

    • Wertverdopplung:

      Der Wert von i wird bei jedem Durchlauf der Schleife verdoppelt.

      i *= 2;
    • Bedingte Anweisung continue:

      Wenn i gleich 4 ist, wird continue aufgerufen und damit die aktuelle Iteration der Schleife beendet (also alle nachfolgende Anweisungen übersprungen) und (da die Schleifenbedingung immer erfüllzt ist) ein neuer Schleifendurchlauf gestartet.

      if (i == 4) { continue; }
    • Bedingte Anweisung break:

      Wenn i größer als 10 ist, wird break aufgerufen und dadurch die Ausführung der Schleife beendet. Als nächstest wird dadurch die erste Anweisung nach der Schleife (System.out.println("Result: " + i);) ausgeführt.

      if (i > 10) { break; }
    • Ausgabe von i:

      Der aktuelle Wert von i wird ausgegeben, es sei denn, er ist 4 oder die Schleife wird beendet, weil i größer als 10 ist.

      System.out.println(i);
  • Abschlussausgabe:

    Nach Beendigung der Schleife wird das Ergebnis von i ausgegeben.

    System.out.println("Result: " + i);
