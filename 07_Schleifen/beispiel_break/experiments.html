Experimente

Ändere die Werte in den Bedingungen auf beliebige Zahlen und versuche vorherzusagen welche Ausgabe das Programm damit erzeugen wird.
Ändere den Quellcode so, dass weder continue noch break verwendet werden. Das Programm soll weiterhin eine Schleife verwenden und dieselbe Ausgabe erzeugen. Die Bedingung für die break-Anweisung kann zum Beispiel als Laufbedingung der while-Schleife umformuliert werden. Die if-Bedingung der continue Anweisung kann so erweiter werden, dass alle folgenden Anweisungen nur dann ausgeführt werden, wenn diese nicht erfüllt ist. 
while (i <= 10) {
    i *= 2;
    if (i != 4) {
        System.out.println(i);
    }
}
