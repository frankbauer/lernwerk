Aufgabe

In dieser Aufgabe werden Sie lernen, wie man statische Methoden und Attribute verwendet, um Methodenaufrufe zu zählen.

Stelle Sie sich vor, dass Sie das Verhalten eines bestimmten Codes überwachen möchten. Dazu soll Ihr Programm zählen, wie oft eine bestimmte Methode aufgerufen wird. In dieser Aufgabe sollen Sie eine Klasse namens Watched implementieren, die diese Funktionalität bereitstellt.

Die Watched-Klasse soll eine statische Methode watchedMethod() enthalten, die jedes Mal, wenn sie aufgerufen wird, einen Zähler um eins erhöht. Sie soll auch ausgeben, dass sie aufgerufen wurde. Des Weiteren soll die Klasse eine Methode printCallCounter() haben, die die Anzahl der Aufrufe der Methode watchedMethod() ausgibt. Setzen Sie die Sichtbarkeit ihrer Methoden und Attribute so restriktiv wie möglich.

Die vorbereitete main-Methode in der Klasse CallCounter enthält bereits eine Schleife, die für eine zufällige Anzahl von Wiederholungen die Methode watchedMethod() aufrufen kann. Ergänzen Sie die main-Methode so, dass nach Ablauf der Schleife die Anzahl der Aufrufe ausgegeben wird. Ihre Ausgabe könnte wie folgt aussehen: 

Method called
Method called
Method was called 2 times
