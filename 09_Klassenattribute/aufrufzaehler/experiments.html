Experimente

Ändern Sie das Programm so ab, dass die while-Schleife in der main-Methode abgebrochen wird, sobald die Methode mehr als dreimal aufgerufen wurde. Achten Sie darauf, dass das Klassenattribut zum Zählen der Aufrufe nur innerhalb der Klasse Watched verändert werden kann. Sie müssen eine Methode einführen, mit der der Wert des Zählattributs gelesen werden kann. 
class Watched {
    public static int getCallCounter(){
        return Watched.callCounter;
    }
    /*...*/
}
Außerdem muss in der while-Schleife die Bedingung angepasst werden: 
if (Math.random() < 0.2 || Watched.getCallCounter() > 3) { break; }
