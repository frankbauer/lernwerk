Erstellen Sie eine zweite Klasse Hauptprogramm mit einer main-Methode. Ändern Sie dort den Steuersatz und geben Sie den brutto Preis einer beliebigen Summe aus. Jede .java-Datei darf immer nur eine öffentliche Klassen enthalten. Wenn Sie eine zweite anlegen muss die öffentliche Sichtbarkeit von Steuer entfernt werden. 
class Steuer { /* ... */ }
 
public class Hauptprogramm {
    public static void main(String[] args){
        Steuer.setMehrwertSteuer(0.1);
        System.out.println( Steuer.berechnePreis(9.99) );
    }
}
