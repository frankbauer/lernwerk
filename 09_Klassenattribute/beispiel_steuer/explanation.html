Erklärung

Die Klasse Steuer dient der Berechnung des Brutto-Preises unter Berücksichtigung der Mehrwertsteuer. Dieses Beispiel konzentriert sich auf die Verwendung von Klassenattributen.

  • Klassenattribut mehrwertSteuer:

    public static double mehrwertSteuer = 0.19;
    Dieses Attribut hält den aktuellen Mehrwertsteuersatz fest. Als static-Attribut gehört es zur Klasse selbst und nicht zu einzelnen Instanzen der Klasse. Es wird initial mit 19% (0.19 als Dezimalzahl) festgelegt.

  • Klassenmethode berechnePreis:

    public static double berechnePreis(double nettoPreis)
    Diese Methode berechnet den Brutto-Preis basierend auf dem übergebenen Netto-Preis, indem der Netto-Preis mit (1.0 + Steuer.mehrwertSteuer) multipliziert wird. Die Nutzung des Klassenattributs mehrwertSteuer ermöglicht die Anwendung des aktuellen Mehrwertsteuersatzes auf alle Berechnungen, die über diese Methode ausgeführt werden.

  • Klassenmethode setMehrwertSteuer:

    public static void setMehrwertSteuer(double prozent)
    Diese Methode erlaubt das Ändern des Mehrwertsteuersatzes. Die Methode überprüft zuerst, ob der neue Satz zwischen 0 und 1 liegt, um ungültige Werte zu vermeiden. Bei einer gültigen Eingabe wird das Klassenattribut mehrwertSteuer entsprechend aktualisiert. Diese Flexibilität ist nützlich, um auf Änderungen der Steuerpolitik reagieren zu können.

  • Hauptmethode main:

    public static void main(String[] args)
    Die Hauptmethode dient als Einstiegspunkt für die Ausführung des Programms. Hier wird die Methode berechnePreis aufgerufen, um den Brutto-Preis für einen spezifischen Netto-Preis zu berechnen und das Ergebnis wird ausgegeben.
