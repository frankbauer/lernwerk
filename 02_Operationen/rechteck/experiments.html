Experimente

Überlegen Sie sich, wie Sie die Werte von zwei gegebenen Variablen (a, b) tauschen können, ohne dafür eine neue Hilfsvariable zu erstellen.

Der Wertetausch ohne eine Hilfsvariable kann durch die Verwendung von arithmetischen Operationen (+, -, *, /, %) realisiert werden.

Code:
a = a + b;
b = a - b; //b = (a + b) - b = a
a = a - b; //a = (a + b) - a = b

Beispiel: a = 1, b = 2
a = 1 + 2 (= 3)
b = 3 - 2 (= 1)
a = 3 - 1 (= 2)
