Erklärung

Dieses Programm berechnet die Fläche und den Umfang eines Rechtecks. Es liest die Maße des Rechtecks aus den Kommandozeilenargumenten, führt die Berechnungen durch und gibt die Ergebnisse aus.

  • Initialisierung der Maße:

    Die Werte der Eingabe (Breite und Höhe) werden über den Parameter args an dein Programm übergeben. Durch Aufrufen der Klassenmethoden getWidth und getHeight aus der Klasse Input werden die Werte aus args als int nutzbar. 

    int width = Input.getWidth(args);
int height = Input.getHeight(args);
  • Berechnung der Fläche:

    Die Fläche des Rechtecks wird berechnet, indem die Breite mit der Höhe multipliziert wird. Dieses Ergebnis wird in der Variablen area gespeichert.

    int area = width * height;
  • Berechnung des Umfangs:

    Der Umfang des Rechtecks wird berechnet, indem die Summe aus Breite und Höhe verdoppelt wird. Dieses Ergebnis wird in der Variablen perimeter gespeichert.

    int perimeter = 2 * (width + height);
  • Ausgabe der Ergebnisse:

    Die berechnete Fläche und der Umfang des Rechtecks werden ausgegeben, um dem Benutzer die Ergebnisse der Berechnungen zu präsentieren.

    System.out.println("Flaeche: " + area);
System.out.println("Umfang: " + perimeter);
