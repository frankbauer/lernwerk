Beschreibung

Input repräsentiert eine Hilfsklasse, die für das Lesen der Benutzereingabe zuständig ist. Die zur Klasse zugehörigen statischen Methoden werden als Input."Methodenname" aufgerufen.

Statische Methoden

  • public static int getWidth(String[] args)
    Liest die Breite aus den übergebenen Programmargumenten args und gibt sie (falls möglich) als Ganzzahl zurück.
  • public static int getHeight(String[] args)
    Liest die Höhe aus den übergebenen Programmargumenten args und gibt sie (falls möglich) als Ganzzahl zurück.
