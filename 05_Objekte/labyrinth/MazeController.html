Beschreibung

Die Klasse MazeController steuert die Bewegung des Spielers im Labyrinth. Sie ermöglicht die Verarbeitung von Befehlen.

Konstruktoren

  • public MazeController(String[] args)
    Erstellt einen neuen MazeController. Die Argumente args können verwendet werden, um die Initialisierung des Controllers anzupassen.

Methoden

  • public void setDelay(int delayInMs)
    Setzt die Verzögerung der Animationsschritte in Millisekunden.
  • public void command(int opCode)
    Verarbeitet den Befehl, der durch den opCode spezifiziert wird. Folgende Bewegungsbefehle Können mit Hilfe von Konstanten aus der Klasse Labyrinth ausgeführt werden:
    • Labyrinth.GO_LEFT: Bewegt den Spieler nach links.
    • Labyrinth.GO_UP: Bewegt den Spieler nach oben.
    • Labyrinth.GO_RIGHT: Bewegt den Spieler nach rechts.
    • Labyrinth.GO_DOWN: Bewegt den Spieler nach unten.
    Wenn ein ungültiger opCode übergeben wird, wird eine Fehlermeldung ausgegeben.
