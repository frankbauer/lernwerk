Experimente

Erstellen Sie einen dritten Roboter mit dem Typ 8. Die Klasse Robot enthält keinen Parameterkonstruktor mit dem sowohl der Typ als auch die Position festgelegt werden können. Der Roboter muss also erstellt und anschließend an die gewünschte Position gesetzt werden. 
Robot robot = new Robot(8);
robot.setLocation(420, 420);
robot.setRoom(livingRoom);
Ändern Sie den Code so, dass wo möglich der Standardkonstruktor verwendet wird, das Verhalten des Programmes ansonsten aber unverändert bleibt. 
public class RoboticConstruct {
        public static void main(String[] args) {
            // Erstellt einen neuen Raum, in diesem Fall das Wohnzimmer.
            Room livingRoom = new Room();
    
            // Erstellt einen neuen Roboter und speichert ihn in der Variable 'robi'.
            Robot robi = new Robot();   //Standrdkonstruktor
            // Setzt die Anfangsposition von 'robi'
            robi.setLocation(690, 410);
            // Weist den Roboter 'robi' dem Wohnzimmer zu.
            robi.setRoom(livingRoom);
    
            // Erstellt einen weiteren neuen Roboter und speichert ihn in der Variable 'anna'
            Robot anna = new Robot();   //Standardkonstruktor
            // Setzt die Anfangsposition von 'anna'
            anna.setLocation(260, 430);
            // Weist den Roboter 'anna' dem Wohnzimmer zu.
            anna.setRoom(livingRoom);
            
            // Erstellt den dritten Roboter vom Typ 8
            // Verwendung des Parameterkonstruktors statt dem Standardkonstruktor,
            // da Typ nicht nachträglich festgelegt werden kann.
            Robot robot = new Robot(8);
            // Setzt die Anfangsposition von 'robot'
            robot.setLocation(420, 420);
            // Weist den Roboter 'robot' dem Wohnzimmer zu.
            robot.setRoom(livingRoom);
    
            // Aktiviert die Bewegung für beide Roboter.
            robi.setMoving(true);
            anna.setMoving(true);
            robot.setMoving(true);
        }
}
