Experimente

Erstelle einen dritten Roboter mit dem Typ 8. Die Klasse Robot enthält keinen Parameterkonstruktor mit dem sowohl der Typ als auch die Position festgelegt werden können. Der Roboter muss also erstellt und anschließend an die gewünschte Position gesetzt werden. 
Robot robot = new Robot(8);
robot.setLocation(420, 420);
robot.setRoom(livingRoom);
Änder den Code so, dass (wo möglich) der Standardkonstruktor verwendet wird, das Verhalten des Programmes ansonsten aber unverändert bleibt.
