Beschreibung

Die Klasse Adapter ist eine Art Fernsteuerungsklasse. Sie ermöglicht es, beliebige Befehle und Werte an verschiedene Geräte zu senden. Die Glühbirnen in unserem Beispiel können die Befehle "hue" (Farbwinkel, [0° ... 360°[ ), "brightness" (Helligkeit, [0.01 ... 1.0]), "saturation" (Sättigung, [0.0 ... 1.0]), "on" oder "off" empfangen und verarbeiten. Jede Adapter-Instanz wird durch die Parameter im Konstruktor genau einem Gerät zugewiesen, das dann mit den verschiedenen Instanzmethoden gesteuert werden kann.

Konstruktoren

  • public Adapter(String ip, short port, String endpoint)
    Erstellt eine neue Instanz des Adapters mit den angegebenen Verbindungsdaten. Die IP-Adresse des Geräts, der Port und der Endpunkt können Sie aus der Abbildung entnehmen.

Methoden

  • public void send(String command, double value)
    Sendet den angegebenen Befehl command mit einem Fließkommawert value an das Gerät. Mögliche Befehle sind "brightness" (Helligkeit, [0.01 ... 1.0]), "saturation" (Sättigung, [0.0 ... 1.0])
  • public void send(String command, int value)
    Sendet den angegebenen Befehl command mit einem Ganzzahlwert value an das Gerät. Mögliche Befehle sind "hue" (Farbwinkel, [0° ... 360°[)
  • public void send(String command)
    Sendet den angegebenen Befehl command ohne weitere Werte an das Gerät.Mögliche Befehle sind "on" oder "off"
