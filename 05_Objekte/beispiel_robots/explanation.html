Erklärung

Dieses Programm simuliert die Erstellung und Steuerung von zwei Robotern in einem Raum. Es demonstriert die grundlegende Nutzung von Objektorientierung, indem es Objekte der Klassen Room und Robot verwendet und deren Methoden aufruft, um Roboter in einem spezifischen Raum zu positionieren und zu aktivieren. Ferner zeigt es die Art und Weise, wie Instanzen durch Variablen referenziert werden.

  • Erstellen eines Raumes:

    Ein neues Room-Objekt wird erstellt, das als "Wohnzimmer" fungiert. Dazu wird der Standardkonstruktor der Klasse Room mittels new aufgerufen. Über die Variable livingRoom kann anschließend auf den Raum zugegriffen werden. Diese ist vom Datentyp Room und enthält die Speicheradresse des neu erstellten Raumes. Auf dem Variablennamen können nun alle Instanzmethoden (Methoden, die kein static in der Methodensignatur aufweisen) über den Punktoperator aufgerufen werden. Zudem kann dieser anderen Methoden übergeben werden, wenn diese einen Übergabeparameter vom Datentyp Room erwarten.

    Room livingRoom = new Room();
  • Erstellung und Positionierung des Roboters 'robi':

    Ein Robot-Objekt wird über den Standardkonstruktor der Klasse Robot erstellt. Die Variable robi referenziert die neue Instanz, das heißt in der Variable ist die Speicheradresse des neuen Roboters gespeichert. Anschließend wird mittels der Instanzmethode setLocation die Position der Instanz auf die Koordinaten (690, 410) gesetzt und die Instanz mit setRoom in das Wohnzimmer aufgenommen. Die Methode setRoom ist eine Instanzmethode der Klasse Robot, weshalb sie auf dem Namen einer Variable vom Datentyp Robot aufgerufen werden muss, hier robi. Laut API benötigt die Methode als Übergabeparameter ein Objekt der Klasse Room, wofür livingRoom übergeben wird. Der Aufruf von setRoom ist notwendig, damit Sie den Roboter im Bild, das das Wohnzimmer zeigt, sehen.

    Robot robi = new Robot();
robi.setLocation(690, 410);
robi.setRoom(livingRoom);
  • Erstellung und Positionierung des Roboters 'anna':

    Ein weiteres Robot-Objekt wird erstellt und eine Referenz darauf in der Variable namens 'anna' gespeichert. Ihre Position wird auf (260, 430) festgelegt und sie wird ebenfalls dem Wohnzimmer zugeordnet.

    Robot anna = new Robot();
anna.setLocation(260, 430);
anna.setRoom(livingRoom);
  • Aktivierung der Bewegung der Roboter:

    Die Bewegung für beide Roboter wird durch Aufruf von setMoving aktiviert, was es ihnen ermöglicht, sich innerhalb des Wohnzimmers zu bewegen.

    robi.setMoving(true);
anna.setMoving(true);
