Experimente

Fügen Sie weitere Roboter an beliebigen Positionen ein. zum Beispiel: 
Robot berti = new Robot();
berti.setLocation(120, 390);
berti.setRoom(livingRoom);
Erstellen Sie einen Roboter mit einem von Ihnen ausgewähltem Typ. Es gibt die Typen 0 bis 11. Die Klasse Robot enthält einen Konstrukter mit dem der Typ festgelegt werden kann. zum Beispiel: 
Robot helena = new Robot(5);
helena.setLocation(400, 400);
helena.setRoom(livingRoom);
