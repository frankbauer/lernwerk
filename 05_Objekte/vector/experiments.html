Geben Sie für jeden Vektor neben der Länge auch dessen L2-Norm aus. Sie können die Instanzmethode norm() verwenden, um die L2-Norm eines Vektors zu berechnen.
Berechnen Sie für jeden Vektor den Winkel, den dieser mit der X-Achse einschließt, und geben Sie diesen aus. Der von den normalisierten Vektoren a → und b → eingeschlossene Winkel kan mit der Formel acos ( a → ⋅ b → ) bestimmt werden. Die Rechenoperation ⋅ ist das Skalarprodukt der beiden Vektoren. Die obige Formel kann mit Hilfe geeigneter Instanzmethoden der Klasse Vec2D wie folgt berechnet werden: 
Vec2D n1 = v1.normalize();
double scalar = Vec2D.XAxis.dot(n1);            
double rad = Math.acos(scalar);
In Java werden Winkel immer in Radianten und nicht in Grad angegeben. Rechnen Sie den Winkel aus der vorherigen Aufgabe in Grad um, bevor Sie den Winkel ausgeben. Um einen Winkel in Grad zu erhalten, können Sie die Methode Math.toDegrees() verwenden. 
double deg = Math.toDegrees(rad);
Schreiben Sie die komplette Winkelberechnung in einer Zeile. 
Math.toDegrees( Math.acos(Vec2D.XAxis.dot(v1.normalize())) );
