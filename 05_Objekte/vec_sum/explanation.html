Erklärung

Dieses Programm zeigt, wie zwei Vektoren im zweidimensionalen Raum addiert werden können. Der Code verwendet die Vec2D-Klasse, um Vektoren zu erstellen und zu addieren.

  • Instanzen der Klasse Vec2D:

    Die Vektoren v1 und v2 sind Objekte (=Instanzen) der Klasse Vec2D. Jede Instanz besitzt ihren eigenen x- und y-Wert, um die Koordinaten ihres Endpunktes im zweidimensionalen Raum bezogen auf ihren Fußpunkt zu repräsentieren. Um eine neue Instanz anzulegen, wird der Parameterkonstruktur von Vec2D mit "new" aufgerufen und es werden zwei Fließkommazahlen für x und y übergeben. In der Beispiellösung ist also der x-Wert des Vektors v1 1.0 und der y-Wert 2.0.

    Vec2D v1 = new Vec2D(1, 2);
Vec2D v2 = new Vec2D(3, 4);
  • Der Punkt-Operator (`.`):

    In Java wird der Punkt-Operator verwendet, um auf Methoden und Attribute eines Objekts oder einer Klasse zuzugreifen. Hier ruft v1.add(v2); die add()-Methode des v1-Objekts auf und verwendet v2 als Argument. Dieser Aufbau des Methodenaufrufs geht aus der API der Klasse Vec2D hervor. Das Fehlen des Schlagwortes static in "public Vec2D add(Vec2D v)" signalisiert, dass es sich um eine Instanzmethode handelt. Folglich muss die Methode auf dem Namen einer Vec2D-Instanz wie z.B. v1 aufgerufen werden und nicht auf dem Klassennamen. Als Übergabeparameter der Methode add ist ebenfalls ein Vec2D-Objekt gefordert, erkennbar an "Vec2D v" in den Klammern nach dem Methodennamen. Hierfür wird im Beispiel v2 übergeben.

  • Die Methode add():

    Die add()-Methode aus der Klasse Vec2D addiert die x- und y-Koordinaten der beiden Vektoren v1 und v2. Sie gibt einen neuen Vektor mit den summierten Koordinaten zurück, der in v3 gespeichert wird.

    Vec2D v3 = v1.add(v2);
  • Die Ausgabe des Ergebnisses:

    Das Statement System.out.println(v3); gibt den Inhalt des Vektors v3 gefolgt von einem Zeilenumbruch aus.
