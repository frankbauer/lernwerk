Erklärung

  • Aufruf von FloatingWorld.show():

    Diese Methode wird aufgerufen, um die FloatingWorld anzuzeigen. Sie ist statisch und gehört zur Klasse FloatingWorld.

  • Erzeugung von Spielerinstanzen:

    FloatingWorld.addPlayer() wird aufgerufen, um einen neuen Spieler in die Welt einzufügen. Das Ergebnis der Methode ist eine Referenz auf ein Player-Objekt.

    Durch die Zuweisung wird diese zurückgegebene Referenz in der entsprechenden Variable (kara bzw. lee) abgelegt. Damit haben wir nun eine Variable über die wir auf die durch addPlayer() erzeugten Objekte zugreifen können. 

    Player kara = FloatingWorld.addPlayer();
Player lee = FloatingWorld.addPlayer();
  • Bewegung eines Spielers:

    kara.moveLeft(); Die Instanzmethode moveLeft() wird auf dem von kara referenzeirten Objekt aufgerufen. Durch den Aufruf wird der Interne Zustand (in diesem Fall die x-Koordinate) des referenzierten Objekts verändert.
