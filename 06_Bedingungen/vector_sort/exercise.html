Aufgabe

In dieser Aufgabe werden Sie lernen, wie man eine einfache Fallunterscheidung vornimmt. Sie werden außerdem Instanzen erstellen und mit deren Methoden arbeiten.

Erstellen Sie zwei 2D-Vektoren (Klasse Vec2D, API im entsprechenden Tab) mit zufällig generierten Werten für die x- und y-Komponente, jeweils im Bereich -25 bis 25 ([-25, 25[). Verwenden Sie zur Generierung der Werte eine passende Methode der Klasse Math.

Berechnen Sie danach den Winkel, den jeder der Vektoren mit der X-Achse aufspannt. Der eingeschlossene Winkel (in Radianten) zwischen zwei normalisierten Vektoren kann mit der Formel acos ( v 1 → ⋅ v 2 → ) berechnet werden. Die Rechenoperation ⋅ ist das Skalarprodukt der beiden Vektoren.

Geben Sie zum Schluss die Vektoren nach diesem Winkel sortiert aus. Beginnen Sie die Ausgabe mit dem Vektor, der zum kleinsten Winkel gehört.
