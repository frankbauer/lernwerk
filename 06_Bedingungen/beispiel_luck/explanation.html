Erklärung

Dieses Programm simuliert den Würfelwurf eines Halblings in einem Rollenspiel, bei dem die besondere Fähigkeit "Halfling Luck" verwendet wird. Diese Fähigkeit ermöglicht es, kritische Fehlschläge (eine 1 auf einem 20-seitigen Würfel) erneut zu würfeln.

  • Initialisierung der Glück-Variable:

    Die halflingLuck Variable wird auf true gesetzt, um anzudeuten, dass der Spieler Glück hat.

    final boolean halflingLuck = true;
  • Bedingung für kein Glück:

    Es wird geprüft, ob der Halbling kein Glück hat (!halflingLuck entspricht halflingLuck==false). Wenn halflingLuck den Wert false hat, zeigt das Programm an, dass der Spieler kein Glück hat. 

    if (!halflingLuck) {
    System.out.println("Du hast kein Glueck!");
}
  • Generierung eines Würfelwurfs:

    Ein Würfelwurf wird simuliert, indem eine zufällige Ganzzahl zwischen 1 und 20 generiert wird. Dieser Wert wird dem Benutzer immer angezeigt.

    int d = (int)(Math.random() * 20) + 1;
System.out.println("Der Wuerfel zeigt: " + d);
  • Überprüfung und Anwendung von Halfling Luck:

    Wenn der Spieler Glück hat (halflingLuck) und (&&) der Würfel eine 1 zeigt (d == 1), wird der Wurf wiederholt und das neue Ergebnis angezeigt.

    if (halflingLuck && d == 1) {
    d = (int)(Math.random() * 20) + 1;
    System.out.println("Aber mit Glueck eine " + d);
}
