Aufgabe

Berechnen Sie die Lösung einer quadratischen Gleichung der Form a x 2 + b x + c = 0 für beliebige Werte von a , b und c . Bestimmen Sie dazu zunächst die Diskriminante D = b 2 - 4 a c .

Ist die Diskriminante negativ, so soll der Text "Keine Lösung" ausgegeben werden. Ist sie 0 , so soll die einzelne doppelte Lösung angezeigt werden ( x = - b 2 a ). In allen anderen Fällen sollen beide Lösungen ausgegeben werden ( x 0 , 1 = - b ± D 2 a ).

Hinweis: Die Wurzel kann mit Math.sqrt(double) bestimmt werden.
